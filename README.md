# 🛒 Retail Sales Analysis System

A comprehensive data analysis project exploring 1,000 retail transactions
to uncover customer behaviour patterns, seasonal trends, and revenue insights
across Electronics, Clothing, and Beauty categories.

---

## 📊 Project Overview

This project performs end-to-end exploratory data analysis (EDA) on a retail
sales dataset using Python. It transforms raw transaction data into actionable
business intelligence through visualisations, statistical summaries, and
data-driven recommendations.

---

## 🎯 Objectives

1. Identify which product categories generate the most revenue
2. Understand gender-based purchasing behaviour differences
3. Determine the most valuable customer age segments
4. Detect monthly and seasonal sales fluctuations
5. Explore how pricing affects purchase volumes
6. Build customer segmentation insights
7. Translate findings into business recommendations

---

## 📁 Dataset Description

**File:** `retail_sales_dataset.csv`
**Records:** 1,000 transactions | **Period:** 2023

| Column | Description |
|--------|-------------|
| Transaction ID | Unique identifier for each transaction |
| Date | Date the transaction occurred |
| Customer ID | Unique identifier for each customer |
| Gender | Customer gender — Male or Female |
| Age | Customer age (18–64 years) |
| Product Category | Electronics, Clothing, or Beauty |
| Quantity | Units purchased (1–4) |
| Price per Unit | Price of a single unit in USD |
| Total Amount | Total transaction value (USD) |

---

## 🔍 Key Findings

- **Electronics** generated the highest revenue at **$156,905** (34.5% share)
- **Female customers** contributed slightly more revenue — $232,840 vs $223,160
- The **46–55 age group** is the highest-spending segment at **$100,690**
- **May** was the peak revenue month at **$53,150**
- **September** recorded the lowest revenue at **$23,620** — a key opportunity
- All three categories have similar average order values (~$443–$467)
